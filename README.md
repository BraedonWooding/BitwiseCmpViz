# bitwisecmp

A nice app to visualise a sequence of bitwise operations.

Based on [BitwiseCmd](https://github.com/BorysLevytskyi/BitwiseCmd).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
